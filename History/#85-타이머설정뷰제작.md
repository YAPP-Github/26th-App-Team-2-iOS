### 주로 사용한 모디파이어 정리

1. `.coordinateSpace(name: scrollViewProxyID)`
-> 스크롤의 offset을 추적하는 모디파이어, 셀이 현재 스크롤 뷰에서 어느 offset에 위치하는지 파악하기 위함 + 이를 통해 offset에 따른 폰트 사이즈, opacity를 조절한다.
2. `@State private var count: Int = 0`
-> count 값을 추적해서 현재 가르키는 값을 표기함
-> scrollPosition에서 중앙에 있는 값을 바꿈

``` swift
    .safeAreaPadding(.vertical, proxy.size.height / 2.0) // 첫 아이템을 중앙에 위치하게 함
    .scrollTargetBehavior(.viewAligned) // 스크롤이 확 일어나지 않게 함
    .scrollPosition(id: .init(get: { count },
        set: { value in
        if let value { count = value }
        }
    ))
```
